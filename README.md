# 🚀 Bootcamp Back-end com Java & QA | DIO + Almaviva Solutions 

Este repositório foi criado para registrar minha evolução ao longo do bootcamp Back-end com Java & QA, oferecido pela [Digital Innovation One](https://www.dio.me/) em parceria com a [Almaviva Solutions](https://www.almavivasolutions.com.br/pt-br/portal/home).

Aqui você encontrará meus estudos, práticas e exercícios desenvolvidos durante o curso, acompanhando meu progresso desde os fundamentos até conteúdos mais avançados.

## 📚 Sobre o Bootcamp 

O programa tem como foco a formação em desenvolvimento back-end com Java, aliado a fundamentos de Quality Assurance (QA), incluindo testes e boas práticas de desenvolvimento.

Ao longo da jornada, são abordados temas como:

- Lógica de programação
- Fundamentos de Java
- Programação Orientada a Objetos (POO)
- Estruturas de dados
- Versionamento com Git e GitHub
- Testes e conceitos de QA
- Automação de testes (introdução)

## 📈 Evolução contínua

Este repositório está em constante atualização.

- Novos conteúdos serão adicionados conforme avanço nos estudos
